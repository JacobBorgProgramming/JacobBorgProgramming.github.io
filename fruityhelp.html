"This browser is not support"

This is a common error code from Google websites. Fruity is supported, just these sites do not know it. Fruity is built on WebKit (Safari) which is supported by these sites.

"When I click some link, they don't open."

This may occur when you (the user) clicks on a link that is supposed to open in a new window or tab. The current solution to this is select and copy the URL/link, click the Fruity icon in the top left, then paste the link into the search bar. The direct link should appear in the search results.

"The browser isn't loading websites."

Check your internet connection. If that isn't the problem, close and reopen Fruity.

"Where's the search bar?"

Click the Fruity logo on the top left and it will appear! This is part of the minimalist design of Fruity.
